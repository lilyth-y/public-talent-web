# 갤러리·활동 등록 방법

비개발자·디자이너도 블로그 쓰듯 갤러리 사진과 활동 글을 등록할 수 있도록 단계별로 정리했습니다.  
Supabase가 설정된 환경에서만 갤러리·활동 게시글을 사용할 수 있습니다.

---

## 1. 갤러리에 사진·글 넣는 방법

갤러리는 **사진만 올려도 됩니다.** 제목과 설명은 원할 때만 입력하면 됩니다.

### 단계

1. 사이트 주소 뒤에 **`/admin`**을 붙여 접속합니다. (예: `https://학부사이트주소/admin`)
2. 이메일과 비밀번호로 **로그인**합니다.
3. 상단 메뉴에서 **갤러리**를 클릭합니다.
4. **사진** 입력란에서 **파일 선택**으로 이미지를 고릅니다.
5. (선택) **제목(한 줄, 선택)** 또는 **설명(한 줄, 선택)**에 내용을 넣습니다. 비워 두어도 저장됩니다.
6. **업로드** 버튼을 누릅니다. 메인 사이트의 「소식·갤러리」 페이지에 바로 반영됩니다.
7. 표시 순서를 바꾸려면 각 사진 카드 아래 **위로** / **아래로** 버튼으로 조정합니다.

### 요약

- 사진만 올려도 됨. 제목·설명은 선택.
- 순서는 **위로** / **아래로** 버튼으로 변경.

---

## 2. 활동 글 쓰는 방법

행사 후기, 세미나 요약, 단체 활동처럼 **한 편씩** 쓰는 글이 활동 게시글입니다.  
**날짜와 제목만 넣어도 저장**되고, 요약·본문·대표 사진은 선택입니다.

### 단계

1. 관리자 로그인 후 상단 메뉴에서 **활동 게시글**을 클릭합니다.
2. **새 활동 추가** 버튼을 누릅니다.
3. **날짜**와 **제목**을 입력합니다. (필수)
4. (선택) **요약(한 줄, 선택)**에 목록·카드에 보일 한 줄 설명을 넣습니다.
5. (선택) **본문(선택)**에 글 내용을 입력합니다. 메모장처럼 여러 줄로 쓰면 그대로 글로 보입니다. 서식(굵게, 링크 등)은 없고 줄바꿈만 반영됩니다.
6. (선택) **대표 사진(선택)**에서 이미지를 고르면 글 상단에 표시됩니다.
7. **저장** 버튼을 누릅니다.
8. 목록에서 **수정** 또는 **삭제**로 기존 글을 관리할 수 있습니다.

### 요약

- 필수: 날짜, 제목.
- 선택: 요약(한 줄), 본문(여러 줄), 대표 사진.
- 본문은 그냥 여러 줄 입력하면 그대로 글로 보입니다.

---

## 3. 활동내역 보는 방법 (외부 공유)

공지·갤러리·활동 게시글을 **날짜순으로 한 목록**에서 볼 수 있는 페이지가 있습니다.  
외부에 "학부에서 뭘 했는지" 보여 줄 때 이 주소만 알려 주면 됩니다.

### 접속 방법

- 사이트 주소 뒤에 **`/community/activities`**를 붙여 접속합니다.  
  예: `https://학부사이트주소/community/activities`
- 또는 메인 사이트에서 **소식·갤러리** 페이지로 들어간 뒤, **「활동내역 (날짜순 전체 보기)」** 버튼을 클릭합니다.

### 화면에 나오는 것

- **날짜 | 공지/갤러리/활동 | 제목(또는 한 줄 설명) | 더보기** 형태로 한 목록에 표시됩니다.
- **더보기**를 누르면 해당 공지(또는 외부 링크), 갤러리 섹션, 활동 글 상세로 이동합니다.

인쇄 시에는 버튼·링크는 숨기고 목록만 깔끔하게 나오도록 되어 있습니다.
