# 공지·소식·갤러리 등록 방법

학부 홈페이지의 **최신 소식**(홈), **공지사항**(학사정보), **소식·갤러리**는 **관리자 페이지**에서 등록·수정할 수 있습니다.  
Supabase를 사용하지 않는 환경에서는 공지만 **JSON 파일**로 수동 등록합니다.

---

## 1. 관리자 페이지로 등록 (Supabase 사용 시)

### 접속

1. 사이트 주소 뒤에 `/admin`을 붙여 접속합니다. (예: `https://학부사이트주소/admin`)
2. **이메일**과 **비밀번호**로 로그인합니다. (관리자 계정은 Supabase 대시보드에서 미리 생성됩니다.)

### 공지 등록·수정

1. 로그인 후 **공지사항** 메뉴를 클릭합니다.
2. **새 공지 추가**로 제목, 날짜, 구분(카테고리), 링크(선택)를 입력해 저장합니다.
3. 목록에서 **수정** 또는 **삭제** 버튼으로 기존 글을 관리할 수 있습니다.
4. 저장하면 홈의 「최신 소식」, 학사정보의 「공지사항」, 소식·갤러리의 「소식」에 바로 반영됩니다.

### 갤러리 등록

1. **갤러리** 메뉴에서 **파일 선택**으로 이미지를 고른 뒤 **업로드**합니다.
2. 제목·설명(선택)을 입력해 저장합니다.
3. 목록 순서는 관리자 페이지에서 조정할 수 있으며, 메인 사이트의 「소식·갤러리」 페이지에 표시됩니다.

### 반영 시점

- 공지·갤러리를 저장하면 **즉시** 메인 사이트에 반영됩니다. 별도 배포 없이 새로고침하면 확인할 수 있습니다.

---

## 2. JSON 파일로 공지만 등록 (Supabase 미사용 시)

Supabase를 설정하지 않은 경우, 공지는 **한 개의 JSON 파일**만 수정하면 됩니다.

### 수정할 파일

- **위치**: 프로젝트 폴더 안 `src/data/notice.json`
- **역할**: 공지·소식 목록을 JSON 형식으로 저장합니다. 이 파일만 바꾸면 홈, 공지사항, 소식·갤러리 화면에 동일하게 반영됩니다. (갤러리는 이 방식으로는 등록되지 않습니다.)

### 파일 형식 (JSON)

파일은 **배열** 형태이며, 글 하나당 **한 개의 객체**입니다.

| 필드     | 설명 | 예시 |
|----------|------|------|
| **id**   | 글 번호 (숫자, 겹치지 않게) | 1, 2, 3, … |
| **title**| 제목 | "2026학년도 수시 모집요강 안내" |
| **date** | 날짜 (YYYY-MM-DD) | "2026-09-01" |
| **category** | 구분 (입학, 학사, 행사 등) | "입학" |
| **link** | (선택) 외부 링크. 비우면 공지사항 페이지로 연결 | "" 또는 "https://..." |

- **link**를 비워 두면: 클릭 시 학부 사이트의 「공지사항」 목록 페이지로 이동합니다.
- **link**에 주소를 넣으면: 클릭 시 해당 주소가 새 창에서 열립니다.

### 예시

```json
[
  {
    "id": 1,
    "title": "2026학년도 수시 모집요강 안내",
    "date": "2026-09-01",
    "category": "입학",
    "link": ""
  },
  {
    "id": 2,
    "title": "학부 정기 학술세미나 개최",
    "date": "2026-08-15",
    "category": "행사",
    "link": "https://www.hs.ac.kr/..."
  }
]
```

- **순서**: 위에 쓴 글이 목록에서 **맨 위**에 보입니다. 새 공지는 배열 **맨 앞**에 추가하면 됩니다.
- **쉼표**: 객체와 객체 사이에는 반드시 쉼표(,)를 넣고, **마지막 객체 뒤에는 쉼표를 넣지 않습니다.**

### 수정 시 유의사항

1. **따옴표**: 제목·날짜·구분·링크는 모두 **큰따옴표(")** 로 감싸야 합니다.
2. **한글/특수문자**: 제목 등에 큰따옴표(")가 들어가면 JSON이 깨질 수 있으니, 가능하면 다른 표현으로 바꾸거나 개발자에게 문의하세요.
3. **저장**: 파일을 수정한 뒤 **UTF-8**로 저장해 주세요.

### 반영(배포) 절차

- 이 파일을 수정·저장하면 **다음에 사이트를 배포할 때** 자동으로 반영됩니다.
- **바로 반영이 필요할 때**: 수정한 `notice.json` 파일을 개발자/관리자에게 전달하고, “배포 부탁드립니다”라고 요청하시면 됩니다.
