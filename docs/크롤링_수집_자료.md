## 4. 전공 홈페이지(Super) – 학과소개

### 학과연혁
- 2021년 공공인재학부 신설, 행정학전공·정책학전공
- 2023학년도 계열제 전환 후 공공인재빅데이터융합학으로 개편, 행정학→공공관리전공트랙, 정책학→공공빅데이터전공트랙
- 2025학년도 경제통상국제공공인재계열로 모집(경제금융, 동아시아통상, 국제관계학과와 함께)
- 2023·24학년도 글로벌공공인재융합계열, 2026학년도 글로벌융합계열 모집. 1학년 말 계열 내 4개 전공(공빅, 경제금융, 동아시아통상, 국제관계) 중 선택

### 공빅 강점
- 행정학 + 빅데이터분석 융합, 문·이과 구분 없음
- 공공관리·공공빅데이터 두 트랙으로 행정학사·빅데이터분석학사 동시 취득 가능

### 졸업 후 진로
- 공무원·공기업 시험, 공기업·민간 빅데이터분석 직무, 복수전공 시 데이터 기반 마케팅/경영 등
- 공공/민간 연구소, 시민단체, 대학원, 스타트업, 프리랜서, 데이터 분석/사이언스팀, 5/7/9급 공무원 등
